## 什么是 Hugo

Hugo 是一个用 Go 语言编写的静态站点生成器，以其极快的构建速度而闻名。

## 为什么选择 Hugo

1. 构建速度极快，毫秒级生成数千页面
2. 基于 Markdown 的内容管理
3. 丰富的主题生态
4. 内置模板引擎
5. 支持多语言

## Hugo 的工作原理

Hugo 将你的 Markdown 内容与模板结合，生成纯静态的 HTML 文件。这意味着：

- 不需要数据库
- 不需要服务器端脚本
- 可以直接部署到任何静态文件托管服务
- 安全性极高

## 核心概念

### 内容组织

Hugo 的内容文件放在 `content/` 目录下，每个页面都是一个 Markdown 文件。

### 模板系统

模板使用 Go 的 `html/template` 包，语法简洁直观。

### 短代码

短代码是 Hugo 的强大功能，允许你在 Markdown 中嵌入复杂的 HTML 组件。
